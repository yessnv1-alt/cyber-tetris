Un motor de juego Tetris 2D moderno y vibrante, desarrollado en un solo archivo con **JavaScript Vanila**, **HTML5 Canvas** y **Bootstrap 5**. Este proyecto combina la mecánica clásica de 1984 con una estética futurista de alto contraste.

---

## 🚀 CARACTERÍSTICAS
- **Diseño Cyberpunk:** Interfaz oscura con efectos neón (Cyan y Pink) y tipografía de alta legibilidad.
- **Sistema Híbrido de Control:** - ⌨️ **Teclado:** Flechas para movimiento, Z/X para rotar, Espacio para caída rápida.
  - 🖱️ **Mouse:** Arrastre lateral para posicionar y clic para rotar.
  - 📱 **Touch:** Botones integrados en pantalla para dispositivos móviles.
- **Dificultad Dinámica:** Selector de velocidad (Novato, Operativo, Cyberpunk y God Mode).
- **HUD Completo:** Sistema de puntuación acumulativa, niveles y temporizador de sesión de 2 minutos.
- **Single File:** Todo el proyecto reside en un solo archivo `index.html` sin dependencias externas pesadas.

---

## 🛠️ TECNOLOGÍAS UTILIZADAS
* **Frontend:** HTML5 (Canvas API) & CSS3 (Custom Variables).
* **Framework UI:** [Bootstrap 5](https://getbootstrap.com/) (Vía CDN).
* **Lógica:** JavaScript ES6+ (Puro, sin librerías de terceros).
* **Despliegue:** GitHub Pages / Vercel.
